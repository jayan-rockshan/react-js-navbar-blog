How to create a responsive navbar using react js

Before start this scenario we have to do installation process

Installation

first you have to install node js then open your command prompt and install npm packages, to do that you can follow below queries otherwise if you not intersted with code this on your IDE you can skip installation proces for now and do this same thing on codesanbox.io
  • npm install
  • npm install -g create-react-app
now we can create a our sample project, here I named my app as a "navbar" if you want you can use anthing whatever you prefer
  • create-react-app navbar
then you have know path what we created folder then change the path on command prompt then you have to type npm start or you can open you IDE then open the folder what we created then open the terminal , on there you can code what we described
  • cd navbar
  • npm start

before start the tutorial you have to know little bit about react hooks and also CSS flex, I suggest this youtube video here if you watch this, you can get a idea about CSS flex
